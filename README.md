# Solidity Voting Smart Contract

A decentralized voting smart contract built using Solidity.

Features:
- Admin adds candidates
- Users can vote once
- Vote counting on blockchain
- Event emitted for each vote

Tech Stack:
- Solidity
- Hardhat
- Ethereum
- GitHub
